# 🌸 HerRecovery

**HerRecovery** is a supportive platform designed to help women struggling with addiction through safe, anonymous, and empowering digital tools.

---

## Features

- 🔐 Authentication (with anonymous option)  
- 💬 Real-time chat / AI companion  
- 📈 Recovery tracker (mood + sobriety)  
- 🚨 SOS emergency support  
- 👭 Safe community space  
- 📚 Recovery resources  

---

## Tech Stack

- Frontend: React (Web), React Native / Flutter (Mobile)  
- Backend: Firebase (Auth, Firestore, Hosting)
